Feature - Resource Plan

What is the Resource Plan Feature in OnePlan?

The OnePlan Resource Planner is a comprehensive resource scheduling and forecasting tool designed to track, optimize, and manage work allocation among various roles within an organization over a specific time frame. It accounts for both committed and planned work hours of all resources within your organization that have been added to OnePlan's Resource Center.

Why Would You Want to Add It?

Adding the Resource Planner to your OnePlan group provides valuable insights into how resources are allocated and utilized over time, enabling better planning, workload distribution, and forecasting. It allows organizations to identify potential overallocated or under allocated resources, promoting efficient allocation of human resources. These insights can help your resources avoid potential burnout, understaffing, and it ensures tasks are assigned to available resources with the right skills at the right time, thus facilitating seamless project management and overall productivity of the group.

Example Use Case

Let's say you have a new project that needs to start in August 2024, and you need a Project Manager and a Developer assigned to work on the project for the next 3 months. Using the Resource Planner, you check for available Project Managers and Developers with the skills you need to deliver the tasks. Using the Resource Planner, as a resource manager, you can update their committed hours and notify the resources – and the Project Manager - of their new assignment. The Resource Planner helps you track their progress and adjust their workload if needed.

Remember the Resource Planner is a tool to help with efficient resource allocation. You should always discuss actual commitments with your team members before assigning them to a new project.
