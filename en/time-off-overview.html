Time Off

Managing time off requests efficiently is crucial for maintaining productivity and employee satisfaction. OnePlan offers robust capabilities to streamline the process of submitting, approving, and tracking time off requests. Whether you are a team member, manager, or resource owner, OnePlan provides tailored functionalities to meet your needs.

Enter a Time Off Request

This article explains how to enter a time off request. Time off in OnePlan will update the resource pool to allow for accurate scheduling.

If you are an approver, see Time off request notifications and approvals .

  1. Go to My Work.

  2. Click Time Off . The Time Off window will open.

  3. Click Add Time Off . The Add Time Off form will open.

  4. Complete the Add Time Off form.

    • Select Resource : Only editable for Owners and Managers. Search for and select the resource to create a time off request for. For Team Members, this field will be auto-filled with your name.

    • Category : Select the time off category. These categories are configured by your OnePlan administrator.

    • Start : Select the time off start date.

    • End : Select the time off end date.

    • Hours Per Day : Select the number of time off hours per day within the set date range.

    • Notes : Enter any notes that the approving manager may need.

    • Status : Only available for Owners and Managers. Select the status of the time off request upon submission. Options include Approved , Requested , and Rejected .

  5. Click Save . The time off request is now submitted. In the Time Off window, you will see a list of all of your time off requests and their statuses.

Time off request notifications and approvals

This article explains the time off request notifications and time off approval.

Time off request notifications

When a user submits a time off request (see Enter a Time Off Request ), approvers are sent notifications of the request. Who receives a notification to approve the time off depends on your organization's OnePlan configuration.

If the environment has "Resource Managers" enabled, OnePlan will send the time off request notifications and emails only to the requesting user's resource manager mapped in the OnePlan configuration. See How do I set Resource Managers?. How do I set Resource Managers?

If the environment only has the "Manager" field configured in the Resource Center, all of the group Owners and the requesting user's "Manager" will receive the time off request notifications and emails.

If the environment does not have "Resource Managers" or the "Manager" field configured, all Owners and Managers in the OnePlan group will receive time off request notifications and emails. Team members, Executives and Contributors will not receive notifications or emails.

Approve or reject a time off request

  1. Go to the time off request.

    • From the request email, click View Request .

    • From the OnePlan Notification Center, click View Request .

  2. Review the time off request and update the Status to Approved or Rejected .

  3. Click Save . The response will be sent to the requesting user via email and a notification.
